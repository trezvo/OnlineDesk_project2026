# OnlineDesk

Десктопное приложение для использования онлайн-досок. Учебный проект по C++.

## О проекте

У пользователя есть возможности по созданию аккаунта, управлению своими досками, записями на них. Приложение поддерживает работу онлайн. На каждой доске можно создавать геометрические фигуры, стрелки, произвольную запись (рисование).

Приложение разделено на клиент и сервер. Клиент — это Qt6 окно, которое запускается как обычная программа. Сервер работает отдельно и принимает подключения. Общаются они по gRPC: весь API описан в двух proto-файлах — `auth.proto` отвечает за регистрацию и вход, `board.proto` за всё, что связано с досками и задачами. C++ код из этих файлов генерируется автоматически при сборке через CMake для клиента, docker compose - для сервера.

На клиенте реализованы следующие окна с соответствующим функционалом: окно регистрации, окно логина, главное меню, окно доски. В главном меню пользователь имеет возможность изменять (создавать/переименовывать/удалять), принадлежащие ему доски, присоединяться к своей, либо к доступной ему чужой доске. При переходе к окну доски пользователь видит все существующие на ней виджеты, а также инструменты по управлению ими. Доступна выгрузка содержиомого доски в PNG, создание снапшота актуального состояния.

Серверный функционал реализован с помощью библиотеки gRPC - фреймворк для сетевого взаимодействия по бинарному протоколу protobuf. Менеджмент досок и авторизация используют unary-interceptor (аналог известных HTTP - запросов). Взаимодействие сервера и пользователей в рамках онлайн-сессии на конкретной доске реализованы с помощью CallbackAPI - обертка над bidirectional-streaming, отличительная технология gRPC, поддерживающая устойчивый клиент-серверный канал связи для передачи данных в обе стороны. Хранение и эксплуатация данных осуществляется с использование СУБД PostgreSQL через ODB - ORM для языка C++. Пароли хешируются через libsodium, пользователи идентифицируются по UUID.

Тесты лежат в отдельной папке и подключаются через CTest.

## Стек

Qt6, C++20, gRPC, Protobuf, libsodium, libuuid.

## Сборка

Зависимости: CMake 3.20+, GCC 11+ / Clang 13+, Qt6, gRPC ≥ 1.44, libsodium, libuuid, ODB, docker compose.

```bash
./script-build.sh # сборка клиента
docker compose build # сборка сервера
```

## Запуск

```bash
docker compose up -d # запуск сервера и базы данных
./script-client-start.sh  # запуск клиента
```

## Тесты

```bash
cd build && ctest --output-on-failure
```
